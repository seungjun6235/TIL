# basic command
>리눅스 기본 명령어 정리

- `pwd`(print working directory)
    - 현재 작업중인 폴더 경로를 출력

- `ls` (list)
    - 현재 작업중인 폴더 내에 있는 파일/폴더를 출력
    - `a`(optional) :
    숨김처리되어있는 파일/폴더까지 출력

 - `cd`(change directory)
    - `cd <폴더이름>` 으로 이동

- `mkdir`(make directory)
    - `mkdir<폴더이름>`
    폴더이름으로 새로운 폴더를 생성
